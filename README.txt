To compile properly run:
- javac MissionariesCannibals.java
- java MissionariesCannibals N M K 
Where fill parameters with: N = Number of Canibals and Missionaries,
    M = Boat size, K = Maximum number of trips boat can do
Terminal will print message if solution is found, 
if yes, it will print every state of the solution,
finally will print the time taken for computer to determine 
if there is a solution or not.
